# 区域经济分析报告生成 Prompt

## 使用说明

将以下 Prompt 复制后，附上数据文件和目标城市名称，即可生成专业的区域经济分析报告。

---

## 📋 通用 Prompt 模板

```
请你扮演一名专业的投融资分析师，基于我提供的区域经济数据文件，为 [城市名称] 生成一份2024年经济与财政深度分析报告。

## 核心分析方法：双基准线对比法

**分析框架**：
1. 计算该城市的**人口占比**（城市常住人口/全省常住人口）
2. 计算该城市的**GDP占比**（城市GDP/全省GDP）
3. 以这两个占比作为基准线，系统对比分析所有财政、债务等关键指标
4. 通过指标占比与基准线的偏离度，揭示结构性特征和问题

## 报告结构要求

### 一、经济总体概况
1. **核心指标基准**：
   - 明确列出人口占比和GDP占比
   - 计算GDP占比/人口占比的效率比值
   - 说明这是后续分析的基准线

2. **经济总量与增速**：报告GDP、增速，与全省对比

3. **产业格局**：三次产业结构及占比

4. **人口规模**：常住人口、户籍人口，分析净流入/流出情况

### 二、GDP与产业结构分析
1. **经济总量分析**：
   - 全省排名
   - 全省占比（计算并说明）
   - 横向对比（与相近城市比较）

2. **产业结构特征**：
   - 三次产业结构比
   - 分别分析第一、二、三产业的增加值、增速、占比
   - 与全省平均水平对比，评价结构特点

### 三、人口分析
1. **人口总量**：报告常住人口及占全省比重
2. **人均GDP**：计算并评价在全省的水平

### 四、财政收支分析（重点使用基准线对比）

1. **财政收入**：
   - 报告一般公共预算收入、增速
   - **全省占比分析**（必须包含）：
     * 占比数值
     * vs GDP占比：偏离多少个百分点
     * vs 人口占比：偏离多少个百分点
   - **结构性评价**：基于偏离度分析财政收入转化效率

2. **收入质量**：税收收入、增速、税收占比

3. **财政支出**：
   - 报告一般公共预算支出、增速
   - **全省占比分析**（必须包含）：
     * 占比数值
     * vs GDP占比：偏离多少个百分点
     * vs 人口占比：偏离多少个百分点
   - 计算人均财政支出，与全省人均支出对比
   - **结构性评价**：分析支出是高配还是低配，解释原因

4. **财政平衡与依赖度**：
   - 财政自给率及全省排名
   - 转移性收入规模及倍数分析

### 五、政府性基金与国有资本经营

1. **政府性基金**：
   - 报告收入、增速
   - **全省占比分析**（必须包含）：
     * 占比数值
     * vs GDP占比：偏离多少个百分点
     * vs 人口占比：偏离多少个百分点

2. **土地财政**：土地出让收入、占比

3. **国有资本**：收入、支出规模

### 六、政府债务与风险分析（重点使用基准线对比）

1. **债务规模**：
   - 报告债务余额、一般债、专项债
   - **全省占比分析**（必须包含）：
     * 占比数值
     * vs GDP占比：偏离多少个百分点
     * vs 人口占比：偏离多少个百分点
   - **结构性评价**：分析债务效率

2. **债务空间**：
   - 债务限额、余额占限额比例
   - **全省占比分析**（必须包含）：
     * 占比数值
     * vs GDP占比、vs 人口占比
   - 评价限额分配的合理性

3. **举债能力分析**：
   - **一般债限额**：
     * 占全省比例
     * vs GDP占比：偏离多少个百分点
     * vs 人口占比：偏离多少个百分点
   - **专项债限额**：
     * 占全省比例
     * vs GDP占比：偏离多少个百分点
     * vs 人口占比：偏离多少个百分点
   - **深度评价**：
     * 分析一般债与专项债限额占比的差异
     * 评价融资结构是否平衡
     * 提出融资策略建议

### 七、综合评价与展望

1. **核心发现：基于基准线的结构性分析**
   - **必须制作对比表格**，包含以下指标：
     * 常住人口、GDP、财政收入、财政支出
     * 政府性基金收入、债务余额、债务限额
     * 一般债限额、专项债限额
   - 表格列：指标名称 | 占全省比例 | vs GDP占比 | vs 人口占比 | 结构特征
   - 总结关键洞察（3-5条）

2. **优势与亮点**（列举5-7条）

3. **挑战与结构性矛盾**（列举3-5条，重点突出基准线对比发现的问题）

4. **投融资策略建议**：
   - 债务融资策略（区分一般债、专项债）
   - 财政收入增长策略
   - 支出效率提升建议
   - 风险防控建议

5. **总体评价**：
   - 用一段话总结城市特征
   - 给出投融资可行性评级（五星制）
   - 列出优势、制约因素、建议

## 输出要求

1. **数据驱动**：所有分析必须基于数据文件中的真实数据
2. **结构性洞察**：充分利用双基准线对比，揭示结构性特征
3. **专业语言**：使用专业、客观、数据驱动的商业报告语言
4. **定量分析**：尽可能进行定量对比，用百分点、倍数等量化表达
5. **深度评价**：不仅要列数据，更要分析数据背后的含义
6. **Markdown格式**：使用标题、列表、表格等格式，确保可读性

## 关键分析原则

✅ **对比优先**：每个关键指标都要与基准线（GDP占比、人口占比）对比
✅ **揭示结构**：重点说明偏离基准线的程度和原因
✅ **评价合理性**：判断该指标是高配/低配/匹配，是否合理
✅ **提出洞察**：基于结构性对比，提出深度洞察和建议

## 特别关注

- 财政支出占比通常会低于人口占比（规模经济效应）
- 一般债限额占比反映公共服务融资能力
- 专项债限额占比反映基建项目融资能力
- 债务余额占比应与人口或GDP占比比较，评估杠杆率
- 土地财政收入变化趋势对财政可持续性的影响
```

## 🎯 核心方法论：双基准线对比法

### 原理
- **人口占比**：反映该城市在全省的人口规模基础
- **GDP占比**：反映该城市在全省的经济体量地位
- **对比分析**：其他指标的占比应与这两个基准相匹配

### 判断标准

| 指标占比情况 | 结构特征 | 典型解读 |
|------------|---------|---------|
| 接近GDP占比 | 与经济体量匹配 | 该指标与城市经济规模相称 |
| 接近人口占比 | 与人口规模匹配 | 该指标按人口平均分配 |
| 显著高于GDP占比 | 超配 | 可能存在过度投入或特殊支持 |
| 显著低于GDP占比 | 低配 | 可能存在资源不足或效率较高 |
| 显著低于人口占比 | 严重低配 | 人均水平低于全省平均 |

### 典型应用场景

1. **财政收入占比 < GDP占比**
   - 说明：财政收入增长未能匹配经济增长
   - 关注：税收征管、产业结构、税收优惠政策

2. **财政支出占比 < 人口占比**
   - 说明：人均财政支出低于全省平均
   - 关注：公共服务水平、规模经济效应、支出效率

3. **一般债限额占比 << GDP占比**
   - 说明：公共服务领域融资能力受限
   - 关注：一般债空间不足，需依靠自有财力

4. **专项债限额占比 ≈ GDP占比**
   - 说明：基建项目融资能力与经济体量匹配
   - 关注：有较好的专项债融资空间

5. **债务余额占比 ≈ 人口占比 << GDP占比**
   - 说明：债务效率高，杠杆率适中
   - 评价：以较低债务支撑较高经济产出

## 🔧 自定义选项

如果需要调整报告侧重点，可以在Prompt末尾补充：

**侧重债务风险分析**：
```
请在报告中特别关注债务风险部分，增加债务率、负债率等指标的深度分析。
```

**侧重投融资可行性**：
```
请重点评估该城市的债务融资空间和项目投资可行性，为投资决策提供依据。
```

**侧重财政可持续性**：
```
请重点分析该城市的财政收入结构、土地财政依赖度和财政可持续性风险。
```

**对比分析多个城市**：
```
请同时生成 [城市A]、[城市B]、[城市C] 的分析报告，并在最后增加横向对比章节。
```
